# Title

josephus survivor porblem

## project description

this repository contains solution for the classical josephus problem. used to determine always surviving position in k-th skip killing of n people arranged in circle.

## problem statement 

its developing a system inorder to always find a surviving position no matter the number of people arranged(N) or position of selection(k). it accepts N and K from user and producecs postion in which survivor is located.

## project objective

build a solution that efficiently handles the problem.

## solution overview

the problem is solved with both simulation and reccurence . it is reccommended to use reccurence because 
it is more efficent relative to simulation one .
